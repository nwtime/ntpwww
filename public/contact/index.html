1. NTP Contact Information

NTP Contact Information

The primary sources of assistance for NTP users are the documentation and information on this website, the Google group, mailing lists, and the FAQ.

The official NTP IRC channel is #ntp on the Freenode IRC network. When posting configuration files or command output on IRC, please use pastebin.com or a similar service. For general IRC usage information, visit irchelp.org .

General Contacts

  • Security Officer: security issues concerning the NTP software implementation must be reported to the Security Officer.
  • Webmaster: please report issues related to this website to the Webmaster.
