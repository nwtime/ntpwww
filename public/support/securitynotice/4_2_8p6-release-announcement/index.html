4.2.8p6 Release Announcement

NTF’s NTP Project has been notified of the following low- and medium-severity vulnerabilities that are fixed in ntp-4.2.8p6, released on Tuesday, 19 January 2016:

  • 2948: Potential Infinite Loop in ntpq / CVE-2015-8158
    • Reported by Cisco ASIG
  • 2945: origin: Zero Origin Timestamp Bypass / CVE-2015-8138
    • Reported by Cisco ASIG
  • 2942: Off-path Denial of Service (DoS) attack on authenticated broadcast mode / CVE-2015-7979
    • Reported by Cisco ASIG
  • 2940: Stack exhaustion in recursive traversal of restriction list / CVE-2015-7978
    • Reported by Cisco ASIG
  • 2939: reslist NULL pointer dereference / CVE-2015-7977
    • Reported by Cisco ASIG
  • 2938: ntpq saveconfig command allows dangerous characters in filenames / CVE-2015-7976
    • Reported by Cisco ASIG
  • 2937: nextvar() missing length check / CVE-2015-7975
    • Reported by Cisco ASIG
  • 2936: Skeleton Key: Missing key check allows impersonation between authenticated peers / CVE-2015-7974
    • Reported by Cisco ASIG
  • 2935: Deja Vu: Replay attack on authenticated broadcast mode / CVE-2015-7973
    • Reported by Cisco ASIG.

Additionally, mitigations are published for the following two issues:

  • 2947: ntpq vulnerable to replay attacks / CVE-2015-8140
    • Reported by Cisco ASIG
  • 2946: Origin Leak: ntpq and ntpdc, disclose origin / CVE-2015-8139
    • Reported by Cisco ASIG, discovered by Boston University

Timeline:

  • 2016 Jan 19: Public release

  • 2016 Jan 17: Early Access Program Release: Premier and Partner Institutional Members
