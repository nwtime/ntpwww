Additional Information

In addition to the implementation documentation, the following resources are available:

  • Reference Library: Whitepapers, RFCs, and other technical documents written by Professor David L. Mills, inventor of NTP.
  • NTP Wiki: Community contributed documentation.
  • NTP FAQ: Frequently Asked Questions about the NTP protocol and software distribution.
  • Public Time Server Lists: Provides an interface to browse or search for a public primary or secondary time server.
  • comp.protocols.time.ntp: Google group. Includes postings from original Usenet newsgroup going back to June, 1990.
